# Quick-CNG

An effective app that solves the problem of waiting time required to visit and check CNG availability or not.

## Overview

Quick-CNG is a Flutter-based mobile application designed to help users quickly find and check the availability of CNG (Compressed Natural Gas) at nearby gas stations. No more unnecessary visits to stations with unavailable CNG!

## Features

✅ **Real-time CNG Availability** - Check CNG availability at nearby stations instantly  
✅ **Location-based Search** - Find CNG stations near your current location  
✅ **Station Information** - View detailed information about CNG stations  
✅ **User-friendly Interface** - Intuitive and easy-to-use design  
✅ **Fast & Reliable** - Optimized performance for smooth user experience  
✅ **Cross-platform** - Available on iOS and Android

## Tech Stack

- **Frontend**: Dart/Flutter (85.6%)  
- **Native Code**: C++ (7.1%), Swift (0.7%), C (0.4%)  
- **Build System**: CMake (5.4%)  
- **Markup**: HTML (0.8%)

## Installation

### Prerequisites
- Flutter SDK (latest version)  
- Dart SDK  
- Android Studio or Xcode (for iOS development)

### Steps

1. Clone the repository:
```bash
git clone https://github.com/yashajaykadav/Quick-CNG.git
cd Quick-CNG
```

2. Get dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```
